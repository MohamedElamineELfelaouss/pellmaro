# Pellmaro — redesign concept

A modern storefront concept for **Pellmaro** (Moroccan suede footwear), plus an orders
back-office. Static site — no build step.

## Deploy (Vercel)
The website lives in **`site/`**. When importing on Vercel, set **Root Directory = `site`**.

- Storefront: `site/index.html`
- Orders dashboard: `site/admin.html`

Details in [`site/README.md`](site/README.md) and [`DEPLOY-AND-PITCH.md`](DEPLOY-AND-PITCH.md).
